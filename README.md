## Технологии

- Vite + React — сборщик и UI-фреймворк
- CSS Modules — изолированные стили компонентов
- Без внешних библиотек для обработки изображений — весь код работы с пикселями написан вручную через Canvas API

## Лаб 1 — Цифровое представление изображения

- Загрузка изображений через кнопку и drag & drop
- Поддержка форматов PNG, JPG и собственного формата курса GrayBit-7 (GB7)
- Отображение загруженного изображения на холсте HTML5 Canvas
- Скачивание в форматах PNG, JPG, GB7
- Реализован декодер и кодер формата GrayBit-7 с нуля:
  - 7-битная градация серого на пиксель (0-127), масштабируется до 0-255
  - Поддержка бинарной маски прозрачности (8-й бит)
  - Big-endian заголовок с сигнатурой GB7, версией и флагами
- Строка состояния с именем файла, форматом, шириной, высотой, глубиной цвета и мегапикселями

└── main.jsx                 — точка входа
